# AttendanceRecord
授業の出席状況を管理できるAndroidアプリ

## Description   
単位を取るためには授業への出席が必須。
自分が授業に何回遅刻・欠席しているか素早く確認することで  
安全に出席して単位を確実に取ろう

## Demo  

準備中

### 現在実装されている機能  
- 授業終了時にスマホへ出欠確認の通知
- 各曜日の授業ごとに出席・遅刻・欠席回数の確認

### 実装予定されている機能
- Google Homeで遅刻・欠席回数の確認
